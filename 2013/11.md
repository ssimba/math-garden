**思考过程**

A B C D E F 6个砝码: 
    
    4 个 1 克的
    1 个 2 克的
    1 个 3 克的

由第一个天平: $A + F < C + D + E$ 可以得出的所有可能性:
    
    1. 1 + 1 < 1 + 1 + 2
    2. 1 + 1 < 1 + 1 + 3
    3. 1 + 2 < 1 + 1 + 3
    得出: A 和 F 是 1 和 2 中的数. B C D E 无法确认.

由第二个天平: $A + C + D < E + F$ 可以得出的所有可能性:

    1. 1 + 1 + 1 < 3 + 2 (当 A 是 1, F 是 2时)
    2. 当 A 是 2, F 是 1 时上式无法成立.
    得出: A = 1, B = 1, C = 1, D = 1, E = 3, F = 2

由第三个天平: $B + C + D = A + F$

    1 + 1 + 1 = 1 + 2
    可以验证上述答案

答案: F 砝码是 2 克, E 砝码是 3 克.
